# Governance Failure and Asset Risk in the HeroGlobal / Numine Ecosystem

This repository documents a factual and verifiable case involving **HeroGlobal**, a blockchain-based game operated within the **Numine ecosystem**.

The purpose of this repository is **risk disclosure**, not accusation.

It presents documented evidence related to:
- selective enforcement and governance inconsistency
- exploit-driven economic distortion
- asset custody and wallet deletion risk
- centralized monetary control in an NFT-labeled system

This material is intended for:
- players
- investors
- blockchain developers
- ecosystem partners
- risk analysts

## Scope
The documentation covers:
- on-chain transaction evidence
- gameplay exploit mechanics and delayed mitigation
- permanent account bans with no economic extraction
- documented cases of wallet deletion
- staff-controlled in-game currency injections

## What this is NOT
- This is not a fraud accusation
- This does not speculate on intent
- This is not a personal dispute

All claims are supported by screenshots, on-chain data, or official announcements.
